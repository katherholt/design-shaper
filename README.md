# What Shape Are You?

Interactive product design self-assessment tool. Map your skills across 6 dimensions, discover your designer archetype, and learn how to adapt in the age of AI.

## Dev

```bash
npm install
npm run dev
```

## Deploy

Push to GitHub and connect to Vercel — zero config needed for Vite + React.

## Stack

- React 18
- Vite
- No external UI libraries — all custom components
